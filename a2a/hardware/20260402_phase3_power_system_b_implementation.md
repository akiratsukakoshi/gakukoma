# Phase 3 Task B：系統B電源構築

**作成日:** 2026-04-02
**作成者:** ClaudeCode（司令塔）
**担当:** Gemini
**前提条件:** Task A1（EEPROM設定）完了後（Task A2と並行着手可）

---

## 目的

11.1V Li-ionパックから LM2596 で 6.0V に降圧し、PCA9685 V+ 端子への DS3218サーボ専用電源を構築する。1000μFコンデンサによるノイズ対策・系統A/BのGND共通接地を確立する。

これが完了することで、Task E（DS3218換装）の電源供給が可能になる。

---

## 電源設計（確定版）

```
11.1V Li-ion パック
    │
    ├──→ LM2596 入力（11.1V）
    │        │
    │        └──→ 6.0V 出力 ─→ [1000μF コンデンサ（並列）] ─→ PCA9685 V+端子
    │
    └──→ TB6612FNG VM端子（11.1V 直結）※ Task D で実施

共通 GND: Pi5 GND ←→ LM2596 GND ←→ PCA9685 GND ←→ TB6612FNG GND
```

---

## 手順

### Step 1: LM2596 電圧設定（バッテリー未接続で準備）

1. LM2596 モジュールを作業台に置く
2. Li-ionパックの XT60コネクタ → LM2596 入力端子に仮接続（まだ本配線しない）
3. LM2596 出力端子にテスターをあてた状態で電源投入
4. LM2596 の**可変抵抗（青いトリマーポテンショメータ）を細いドライバーでゆっくり調整**
5. テスター表示が **6.0V（± 0.1V）** になるまで調整
6. 電源を切断し、テスターを外す

> **注意**: 初回調整時は出力電圧が高い場合がある。時計回りで下がる / 反時計回りで上がるが、モジュールによって逆の場合もある。ゆっくり回してテスターで確認すること。

### Step 2: 1000μF コンデンサの挿入

1. LM2596 の出力端子から PCA9685 V+ 端子へ向かう配線の途中に 1000μF 電解コンデンサを **並列** に接続する
   - **コンデンサのプラス脚（長い方）**: 6V ライン（LM2596 出力 +）
   - **コンデンサのマイナス脚（短い方）**: GND ライン
   - **極性を必ず確認すること（逆接続は破損・発火の原因）**
2. コンデンサのリード線が他の配線やケースと接触しないようにテープや熱収縮チューブで保護

### Step 3: PCA9685 V+ への接続

1. LM2596 出力（6.0V）→ コンデンサ並列点 → PCA9685 の **V+ 端子** に接続
2. **注意**: PCA9685 の VCC（ロジック電源）は引き続き Pi5 の 3.3V（Pin1）から取得する。V+ とは別端子。

### Step 4: GND 共通接地

以下を全て共通の GND ポイントで接続する（ジャンパーワイヤまたは配線で結線）:

| GND 接続先 | ポイント |
|---|---|
| LM2596 GND（出力側） | 共通GND |
| PCA9685 GND | 共通GND |
| Li-ionパック GND（マイナス端子） | 共通GND |
| Pi5 GND（Pin6 または任意のGNDピン） | 共通GND |
| TB6612FNG GND（Task D で追加） | 共通GND（今後ここに合流） |

> **重要**: GND が共通接地されていないと、GPIO制御が誤動作したり、I2Cが不安定になる。必ず全て結線すること。

### Step 5: 電圧・導通の最終確認

1. 系統B（Li-ionパック）を接続した状態でテスターで計測:
   - LM2596 入力電圧（Li-ionパック生電圧）: 10.5〜12.6V（充電状態に依存）
   - LM2596 出力電圧: **6.0V ± 0.1V**
   - PCA9685 V+ 端子電圧: **6.0V ± 0.1V**

2. テスターの導通モードで GND 共通接地を確認:
   - Pi5 GND（Pin6）↔ PCA9685 GND → 導通すること
   - Pi5 GND（Pin6）↔ Li-ionパック GND → 導通すること
