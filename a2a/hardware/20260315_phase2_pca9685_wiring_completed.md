## 1. 実施内容サマリ
指示書に従い、Raspberry Pi 5 と PCA9685 サーボドライバの I2C 接続、および SG90 サーボモーター 2 基の配線と動作確認を実施した。

## 2. 完了条件の確認結果
- [x] **I2C 有効化:** `raspi-config` により有効化を確認済み。
- [x] **デバイス検出:** `i2cdetect -y 1` を実行し、アドレス `0x40`（PCA9685）および `0x70`（All Call）の応答を確認。
- [x] **ライブラリ導入:** `adafruit-circuitpython-pca9685` 等を `--break-system-packages` フラグを使用してインストール完了。
- [x] **動作テスト:** `/tmp/servo_test.py` を実行し、パン軸（ch0）およびチルト軸（ch1）が意図通りに駆動することを確認。

## 3. 発生した問題と対処
- **GNDピンの重複:** 指示書の Pin 6 が Phase 1 のアンプで使用済みであったため、**Pin 9 (GND)** を代替として使用。共通接地ルールに基づき動作に問題ないことを確認。
- **Python環境エラー:** `pip3` インストール時に `externally-managed-environment` エラーが発生。本機はロボット専用環境であるため、OS側の制約を承知の上で `--break-system-packages` を適用して解決。

## 4. 技術仕様および申し送り
- **電源構成:** V+（サーボ電源）は RPi 5 の 5V ピン（Pin 4）から供給。現時点の SG90×2 構成では安定動作している。
- **チャンネル割り当て:**
    - `ch0`: パン（左右）
    - `ch1`: チルト（上下）
- **サーボ方向（Antigravity 向け）：**
    - 角度を大きく（135°方向）すると：パンは **[左]** に回転
    - 角度を大きく（110°方向）すると：チルトは **[下]** に回転
    - ※ `look_at_user()` 実装時の比例制御ゲイン設定の参考にすること。

## 5. 結論
ハードウェア側の配線および基本動作確認はすべて完了した。本報告書をもって Phase 2 の物理セットアップ工程を終了とする。
