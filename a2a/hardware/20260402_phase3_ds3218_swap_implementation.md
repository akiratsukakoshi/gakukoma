# Phase 3 Task E：DS3218換装・アルミパンチルト台座移行

**作成日:** 2026-04-02
**作成者:** ClaudeCode（司令塔）
**担当:** Gemini
**前提条件:** Task B（系統B電源構築・PCA9685への6V供給確認）完了後

---

## 目的

既設の SG90×2 + プラスチック台座を取り外し、DS3218×2 + アルミ合金製パンチルト台座に換装する。PCA9685への電源が XL4015 経由 6V になっていることを確認した上で実施する。

---

## 重要事項

- DS3218 は消費電流が大きく、ラズパイ直接給電は **厳禁**
- PCA9685 V+ に XL4015 の **6.0V** が供給されていることを換装前に必ずテスターで確認すること
- サーボ制御ソフト（`pan_tilt.py`）はDS3218でも変更不要（同じ PWM プロトコル）

---

## 手順

### Step 1: 現行台座の取り外し

1. Pi5 の電源を切った状態で作業
2. SG90 のコネクタを PCA9685 の ch0（パン）・ch1（チルト）から抜く
3. Webカメラを台座から取り外す（ケーブルは傷つけないよう注意）
4. SG90×2 をブラケットから取り外す
5. プラスチック台座をシャーシまたは搭載面から取り外す

### Step 2: アルミ合金製パンチルト台座の組み立て

1. DS3218 を台座のパン軸ブラケットに取り付ける
   - 取付ネジは付属品を使用。締めすぎに注意（アルミ製でもネジ穴を痛める）
2. DS3218 を台座のチルト軸ブラケットに取り付ける
3. 各サーボにラベルを貼る（**「パン ch0」「チルト ch1」**）

### Step 3: サーボホーンのセンター位置出し

サーボを90°（中央）にセットしてからホーンを取り付けることで、後の調整が楽になる。

**手順:**

1. Pi5 を起動（UPS HAT 駆動 or ACアダプタ接続）
2. DS3218 のコネクタを PCA9685 の ch0 に仮接続（まだ台座には取り付けない）
3. 以下のコマンドでセンター位置（90°）を指定:

```bash
/home/tukapontas/gakukoma/tools/set_pan_tilt.sh 90 90
```

4. サーボが動いて停止した位置が **90°（センター）** の位置
5. その位置でサーボホーンを取り付ける（ホーンがまっすぐ正面を向くように）
6. ch1 のチルトサーボも同様に実施

### Step 4: 台座への最終取り付け

1. センター出しが完了したDS3218を台座のパン軸・チルト軸に固定
2. Webカメラをアルミ台座の最上部に取り付け（M3ネジまたは両面テープ）
3. DS3218 のコネクタを PCA9685 の正式チャンネルに接続:
   - **パンサーボ → ch0**
   - **チルトサーボ → ch1**

### Step 5: 電源供給の確認

テスターで PCA9685 V+ 端子を計測:
- **期待値**: 6.0V（± 0.1V）
- 非対応電圧が出ている場合は XL4015 の設定を確認（Task B 参照）

### Step 6: 動作確認

```bash
# センター（正面）に移動
/home/tukapontas/gakukoma/tools/set_pan_tilt.sh 90 90

# パン左端（最大左）に移動
/home/tukapontas/gakukoma/tools/set_pan_tilt.sh 170 90

# パン右端に移動
/home/tukapontas/gakukoma/tools/set_pan_tilt.sh 10 90

# チルト上（最大上）に移動
/home/tukapontas/gakukoma/tools/set_pan_tilt.sh 90 60

# チルト下（最大下）に移動
/home/tukapontas/gakukoma/tools/set_pan_tilt.sh 90 180

# センターに戻す
/home/tukapontas/gakukoma/tools/set_pan_tilt.sh 90 90
```

各コマンドでサーボが期待通り動くことを確認する。

> **カメラ向きの方向について**: 換装後、パン方向（左右）やチルト方向（上下）が従来と逆になる場合がある。その場合はソフト側（`config.yaml` の `pan_gain`/`tilt_gain` の符号）で対応するため、**完了報告に方向情報を必ず記録すること**。
